IEToEdgeWarning.html

Example to Show IE Logo & Link via JQuery Animated Window

This example (when opened in IE) automatically shows a warning to users that IE is not supported by this website and recommends they open this site in Edge (or download Edge). This example uses JQuery and an animated Div [drawer] at the top of the screen.

Here is the required code added to the page (which you need to add to any page using this project):


<script src="https://code.jquery.com/jquery-3.5.1.min.js"></script>
<script src="../iewarning.js"></script>
<style>
    #div-slidedown{
        padding:10px;
        display:none;
        text-align:center;
        background-color:white;
        border:solid 1px black;
    }
</style>

Here is the code that activates this screen:


<body onload="ieWarning('Replace this text with some warning you want to show users.', true, 'edge', 50)">
    <div id="div-slidedown"></div>
